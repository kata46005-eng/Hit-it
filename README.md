# Hit-it
Цель игры в том, чтобы поймать приз несколько раз, а если точнее три. В игре есть счётчик раз, сколько раз вы дотронулись до приза. Есть несколько врагов, после соприкосновения с ними игра закончится, но вы всегда можете попробовать снова. Движение производится с помощью кнопок 'Right', 'Up', 'Down', и 'Left'. Удачи!
Поделиться моим проектом можно по ссылке: https://learn.algoritmika.org/share/59293655
